# Newtrino

A Simple Flat-File Content Managing System

```console
npm install gulp-cli -g
npm install gulp -D
npm install
```

```console
npm start
```

## Upgrade

Replace the following directories and files of your install to ones in ./dist.

* admin/
* core/
* index.min.js
* index.min.js.map
* index.php
* login.php
